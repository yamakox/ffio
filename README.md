# ffmpeg_frame_io

## Install from GitHub

```bash
pip3 install git+https://github.com/yamakox/ffmpeg_frame_io.git
```

## Install from source code

```bash
pip3 install -e .
```

## Examples

```bash
cd examples
python3 frame_writer_sample.py
python3 frame_reader_sample.py
```
