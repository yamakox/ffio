# ffmpeg_frame_io

## Install from GitHub

```bash
pip3 install git+https://github.com/yamakox/ffmpeg_frame_io.git
```

## Install from source code

```bash
pip3 install -e .
```

## Build

```bash
python3 -m venv venv
. venv/bin/activate
pip3 install -r requirements.txt
python3 -m build
```

## Examples

```bash
cd examples
python3 frame_writer_sample.py
python3 frame_reader_sample.py
```
